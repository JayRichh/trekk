Fallback Image Directory

This directory contains fallback images that are used when external image APIs are unavailable.

Required fallback images:
- trail-default.jpg
- mountain-default.jpg
- forest-default.jpg
- hiking-default.jpg

For production deployment, please add these images to this directory.
These should be generic hiking/trail images that can be used as fallbacks.

Recommended size: 400x300 pixels
Format: jpg or webp
